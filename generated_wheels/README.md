# Generated Wheels

All Cython/C extensions are pre-compiled — no C compiler or Cython is needed to install.

## Installing from a wheel

```bash
pip install classixclustering-1.4.4-cp312-cp312-manylinux_2_24_x86_64.manylinux_2_28_x86_64.whl
```

No C compiler or Cython is required when installing from a wheel.

## Building Windows Wheels

Windows wheels can be generated using the `build_windows_wheels.yml` GitHub Actions workflow,
which builds and tests wheels on `windows-latest` for Python 3.9–3.13.
